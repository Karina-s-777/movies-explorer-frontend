 ## Директории

`/components` — папка с компонентами 
`/images` — папка с изображениями  
`/models` — папка со шрифтами
  
Остальные директории вспомогательные, создаются при необходимости разработчиком

## Запуск проекта

`npm start` — запускает сервер
